Notable Projects

Dead Reckoning

An offline location-tracking app for Android

Many modern conveniences rely on our ability to accurately pinpoint location. GPS makes this job much easier, but the system isn't perfect. There are many places where GPS signal is restricted, from underground tunnels, to inside buildings, and even outdoors if near large structures.

Dead Reckoning is an Android app that offers a solution. Using built-in smartphone sensors to track location, the app removes the reliance on GPS and makes the tracking process self-sufficient and offline.

USnoozeULose - The Unforgiving Alarm Clock Capital One Hackathon

An Android alarm clock app for motivating heavy snoozers to wake up on time.

The alarm clock utilizes a number of techniques to get users up and about, including a custom mini-game which must be completed before turning off the alarm, and even a small financial penalty for excessive snoozing (implemented using CapitalOne's Nessie API).

Users are able to easily pick which feature(s) they would like to use to customize their alarm experience.

Rock, Paper, Scissors, Leap HackRU 2016

A web app employing the Leap Motion Controllers to play a game of Rock, Paper, Scissors over the web.

Uses Leap Motion controllers to track and analyze hand movement and determine hand gestures (rock, paper, or scissors). The app communicates game data over a backend Node.js server on a Heroku dyno, allowing for a wireless and real-time game of Rock, Paper, Scissors over the web.

Adele - SMS Translator MHacks: Refactor 2016

A service to translate between written languages purely via SMS. Especially useful in areas where a data connection may be unavailable or prohibitively expensive -- great for use while traveling or when in a pinch.

The service is run on a Node.js backend server hosted on a Linode Ubuntu box. It uses the Google Translate API to translate between languages, and uses the Twilio API to send and receive SMS messages.
